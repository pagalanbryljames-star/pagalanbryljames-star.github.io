Locker Management Sytem

The Locker Management System is a simple computer program made using Java to make managing lockers easy and organized. It allows clients to sign up and log in to access their lockers, select available lockers, view balance, and view payment history, while administrators can add, view, update, or delete locker information. The program also shows which lockers are free or being used based on availability or who is using them. It uses basic Java tools and ideas, like storing data, making decisions, and handling information, to work smoothly.
